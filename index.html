使用说明

本页面将带你从“第一次进入”到“AI 智能诊断—AI 加速练习—专项—错题消除—计划复盘”的完整路径，帮助你用更少时间提分。

回车跳到下一处匹配；再次回车继续跳转。

目录

1. 推荐学习主流程（新手必看）

  1. 登录并选择学员：先完成“驾校账号登录”，再通过扫码/账号密码/调试入口绑定学员。
  2. 设置训练上下文：在顶部设置城市、车型、科目，系统据此显示个性化学习环境。
  3. 先做100题智能诊断：进入「AI 智能诊断」完成一次45分钟诊断，建立初始画像。
  4. 看报告并执行建议：关注预测得分、薄弱知识点、错因分布、ROI提分清单。
  5. 日常走“AI 今日学习任务”：弱项专项20题 → 到期复习 → 快速模考（首页自动打勾）。
  6. 进入 AI 加速练习：系统按高频、掌握度、遗忘风险、错因策略动态出题。

2. 登录与首页怎么用

登录流程（新）：系统支持“驾校登录 + 学员登录”。每个学员的数据独立保存，切换学员不会串数据。

训练上下文（新）：可设置城市/车型/科目，显示在顶部并参与学习引导文案。

全局主按钮：首页右侧有一个主按钮，会根据你的状态自动变成：

  • 未诊断：启动 AI 100题诊断（推荐）
  • 诊断进行中：继续诊断（x/100）
  • 诊断已完成：开始 AI 加速练习

AI 今日学习任务（新）：首页会显示当日任务进度与下一步按钮，点击可直接进入对应模式。

今日目标卡片：展示今日完成/目标、今日正确率、总进度。目标会根据考试日期、剩余题量与复习负载动态计算。

预约考试：点击“预约考试”选择日期，会写入本地并影响计划与诊断页显示。

3. AI 智能诊断（100题+45分钟）

开始/继续/重置：诊断支持中途退出，下次继续；也支持重新开始（清空本次诊断）。

倒计时：诊断中显示45分钟倒计时。时间到会自动结束并生成报告。

显示解析开关：右上角“显示解析”可切换两种模式：

  • 开启：答题后显示解析（更适合边做边学）。
  • 关闭：不显示解析，直接下一题（更像真实考试）。

组卷逻辑：先覆盖主要知识点，再优先抽取高价值题（高频/薄弱/待复习）。

4. 诊断报告怎么读

预测得分（加权）：按知识点掌握度并结合题库考频权重计算，更能反映“高频短板”对分数的影响。

错因分布：基于你标记的错因（记忆/理解/粗心/陷阱）汇总，判断是知识短板还是方法问题。

薄弱知识点TOP：显示掌握度最低的知识点，支持一键进入专项练习。

ROI提分清单：综合“薄弱程度 × 频率权重”，优先练最可能提分的模块。

行动建议：可直接跳转到专项、AI 加速练习、错题冲刺或模拟考试。

5. 练题页怎么用（顺序/智能/复习/错题/专项）

模式切换：右上角按钮会直接切换题池，不只是改标签。

  • 顺序：支持“未做优先 / 全量顺序”两个子模式。
  • 智能：按高频、掌握度、遗忘风险、错因策略动态推荐。
  • 复习：只练到期复习题（1/3/7/14天间隔）。
  • 错题：只练错题本中的题。
  • 专项（自动触发）：从诊断报告点击“去专项练习/一键串练”进入专项队列。

智能锁定：可设置同知识点连续训练题数（5/10/20/30/40/50，默认10）。

AI 训练小结：每组结束展示正确率、掌握度和用时，并可继续下一组。

粗心关键词提示：当“粗心”占比高时，题干会高亮“不/必须/不得/严禁”等关键词。

收藏：点击“收藏”将题目加入收藏列表（用于复习）。

笔记：每题可写笔记（口诀/易错点），保存到本地浏览器。

专项错题回插：在专项模式中，做错的题会在1-2题后再次出现一次，用于短间隔强化。

专项小结：专项最后一题答完后会出现“专项小结”按钮，展示完成题数/正确率/掌握度，并可继续下一步。

6. 错题本怎么用

进入方式：左侧菜单「错题本」。

自动收录：练题答错会自动加入错题本。

消除规则：同一道错题连续答对2次会自动移出（答错会重置计数）。

错题重练：点击“错题重练”进入错题队列。

清空错题本：支持一键清空当前学员错题本（用于重置冲刺前状态）。

错因选择（新）：答错后会出现“错因选择”（记忆/理解/粗心/陷阱）。建议选择后再继续，系统会在诊断报告里汇总“错因分布”，并给出更贴合你的建议。

7. 今日计划、冲刺与题库页

今日计划：根据考试日期、剩余题量、近3天正确率、到期复习量动态计算今日目标，并按倒计时分阶段（基础/平衡/冲刺/临考）。

设置面板（已可用）：点击顶部齿轮可调整“智能锁定题数、每日新题/复习/总题上限、阶段阈值”。保存后会立即更新首页“今日目标”和「今日计划」页。

默认参数：新题上限160、复习上限120、总题上限200；阶段阈值为基础>14天、平衡7-14天、冲刺4-7天、临考≤3天。

调参建议：时间少可先降“新题上限”并提高复习占比；基础薄弱可提高“基础期阈值”；若出现“无法按时完成”风险提示，可延后考试日期或提高每日练题时长。

考前冲刺（更新）：冲刺入口统一按模拟考试流程。薄弱知识点专项模拟会按该知识点实际题量组卷（上限100题），不会强行补满。

题库（已优化）：支持搜索（题干/ID）、按状态/题型/知识点/收藏筛选、分页浏览，并支持“AI 生成练习（N题）”和“将筛选结果加入收藏”。

题库视图切换：支持“列表”与“按知识点”两种视图，可查看掌握度、题量、未掌握数并一键专项。

学习模式选择：在「AI 个性学习」页支持「顺序学习 / 随机练习 / 专项练习 / 错题练习」。

顶部栏按钮：问号跳转使用说明；铃铛为占位提示；齿轮可进入学习设置。

8. 数据存储与隐私（重要）

本系统是纯静态原型：所有数据保存在你的浏览器 localStorage（不会上传服务器）。

换浏览器/清缓存会丢失：如需跨设备同步，需要后续接入账号系统与服务端存储。

9. 常见问题（FAQ）

Q：为什么智能模式会跳题？ A：智能模式会优先推送“最值得练”的题（高频/薄弱/遗忘风险/错因策略），不是按顺序。

Q：冲刺页为什么有时不是100题？ A：薄弱知识点专项模拟按该知识点实际题量组卷，上限100题，不强行补题。

Q：我想清空学习记录怎么办？ A：见 README 里的“如何清空本地数据”，或在浏览器设置里清空站点数据。

Q：切换学员后数据为什么不同？ A：每个学员数据独立快照存储，这是预期行为。
