使用说明（非常详细）

本页面将带你从“第一次进入”到“诊断—加速—专项—错题消除—计划复盘”的完整路径，帮助你用最少时间提分。

回车跳到下一处匹配；再次回车继续跳转。

目录

1. 推荐学习主流程（新手必看）

  1. 设置考试日期：去「首页」点击“预约考试”选择日期（影响今日目标与学习节奏）。
  2. 开始100题智能诊断：进入「智能诊断」点击“开始诊断”。这相当于一次模拟考试（45分钟倒计时）。
  3. 查看诊断报告：完成后会生成“预测得分（加权）/薄弱知识点TOP/提分清单ROI”。
  4. 一键串练ROI专项：在报告里点击“一键串练ROI专项（5×20题）”，系统会按提分潜力从高到低带你做专项。
  5. 回到智能加速：每个专项结束会弹出“专项小结”，点“继续（下一步）”进入下一专项或回到智能加速练习。
  6. 错题消除：错题会进入错题本，同一道错题连续答对2次会自动移出。

2. 首页怎么用

全局主按钮：首页右侧有一个主按钮，会根据你的状态自动变成：

  • 未诊断：开始100题诊断（推荐）
  • 诊断进行中：继续诊断（x/100）
  • 诊断已完成：开始智能加速练习

今日目标卡片：展示今日完成/目标、今日正确率、总进度。目标会根据考试日期和剩余题量动态计算。

预约考试：点击“预约考试”选择日期，会写入本地并影响计划与诊断页显示。

3. 智能诊断（100题+45分钟）

开始/继续/重置：诊断支持中途退出，下次继续；也支持重新开始（清空本次诊断）。

倒计时：诊断中显示45分钟倒计时。时间到会自动结束并生成报告。

显示解析开关：右上角“显示解析”可切换两种模式：

  • 开启：答题后显示解析（更适合边做边学）。
  • 关闭：不显示解析，直接下一题（更像真实考试）。

题量说明：当前为Mock题库，题目不足100时会循环补齐用于流程演示。

4. 诊断报告怎么读

预测得分（加权）：按知识点掌握度并结合题库考频权重计算，更能反映“高频短板”对分数的影响。

薄弱知识点TOP10：掌握度最低的10个知识点。

提分清单TOP5（ROI）：综合“薄弱程度×考频权重”，优先练这5个最划算。

一键串练ROI专项：会按ROI从高到低，依次做5个专项，每个专项20题（专项结束有小结）。

5. 练题页怎么用（顺序/智能/错题/专项）

模式切换：练题页右上角可切换：

  • 顺序：按题库顺序练习。
  • 智能：系统优先挑“高频+低掌握度+久未练”的题，答完会自动跳到更值得练的题。
  • 错题：只练错题本中的题。
  • 专项（自动触发）：从诊断报告点击“去专项练习/一键串练”进入专项队列。

收藏：点击“收藏”将题目加入收藏列表（用于复习）。

笔记：每题可写笔记（口诀/易错点），保存到本地浏览器。

专项错题回插：在专项模式中，做错的题会在1-2题后再次出现一次，用于短间隔强化。

专项小结：专项最后一题答完后会出现“专项小结”按钮，展示完成题数/正确率/当前掌握度，并可继续下一步。

6. 错题本怎么用

进入方式：左侧菜单「错题本」。

自动收录：练题答错会自动加入错题本。

消除规则：同一道错题连续答对2次会自动移出（答错会重置计数）。

错题重练：点击“错题重练”进入错题队列。

7. 今日计划与题库页

今日计划：根据考试日期与剩余题量计算今日目标，并统计今日完成/正确率。

题库：展示每题状态（未做/已掌握/未掌握）并可跳转到指定题。

8. 数据存储与隐私（重要）

本系统是纯静态原型：所有数据保存在你的浏览器 localStorage（不会上传服务器）。

换浏览器/清缓存会丢失：如需跨设备同步，需要后续接入账号系统与服务端存储。

9. 常见问题（FAQ）

Q：为什么题目会重复？ A：当前是Mock题库，用于演示流程；真实题库接入后将不重复。

Q：我想清空学习记录怎么办？ A：见 README 里的“如何清空本地数据”，或在浏览器设置里清空站点数据。

Q：智能模式为什么会跳题？ A：智能模式会优先推送“最值得练”的题（高频短板），所以不是按顺序。
